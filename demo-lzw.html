LZW Compression

History

LZW (Lempel-Ziv-Welch) is a universal lossless data compression algorithm. It is easy to demonstrated for text, but the compression algorithm can be applied to other data types, such as images. Notably, this algorithm is part of the GIF image format. It is also used in the Unix shell program "compress," which has since been deprecated in favor of "DEFLATE," which uses the LZ77 algorithm. The patent for LZW was filed in 1983 and expired in 2003. The compression ratio for sufficiently large English text files is about 50%.

LZW Algorithm

The basic premise of this compression algorithm is that a dictionary is constructed, which is built from substrings that have previously been seen when parsing the file. The key in the dictionary is the substring and the value is an increasing index. Initially, the dictionary is filled with all usable characters and a substring is set as the first character of the data. The algorithm checks whether the substring concatenated with the next character of the data is in the dictionary. If it is, then this new concatenated string becomes the new substring and nothing is outputted. If it isn't, then the code for the substring is outputted, the convatenated string is added to the dictionary, and the new substring is set as the next character. This is repeated for the rest of the data.

About this Demo

This implementation uses standard UTF-8 characters 0-255 for the initial dictionary. Enter any amount of text into the text area on the right and click the "Encode" button. The entries in the initial dictionary are hidden, and result of each character is shown in the table on the left. The sequence mapping section shows how the input text is coded by reusing substrings saved in the dictionary. Hovering over a substring in this section will highlight the corresponding dictionary entry and index. The decimal output section at the bottom shows a list of decimal values which represents the compressed output data. This would normally be in binary, but it is more convenient to read decimal.
